How to use the site:

1) The site is still in development phase.

2) Search feature works only for NBA and for teams listed in the NBA dropdown. Reason: I don't want to exhaust my API call limit during testing. So I have cached the results by implementing a Caching Service. The cache currently has only the listed teams.

3) To Search, please select NBA and type any one of the teams listed. You can type patial team names but they have to be complete words.

4) The Search logic works as follows:
a) Take the entered string and convert it to an ID by looking up the cache
b) Query the SportRadar API using the id
c) Note: My API i.e. SportRadar doesn't have the feature of getting details from name. So I should have a local cache that converts name to ID. For example, you can search for "Celtics" and expect to see the Boston Celtics page

4) I'm yet to implement live scores, results, videos, fixtures and blogs
